מה נלמד השבוע

  1. החלקיק の — קישור שמות ושייכות ("של", "מסוג").
  2. מספרים: 1–10,000 ואיך לבנות מספרים מורכבים.
  3. שאלת מחיר עם いくら ("כמה עולה") והמטבע 円（えん）.

בסוף כל נושא יש שאלות תרגול ותרגילי תרגום, ובסוף השבוע — תיבות בחינה לאוצר המילים ולקאנג'י.

נושא 1

החלקיק の — קישור שמות

החלקיק の מחבר שני שמות כשהראשון מתאר, מסביר או שייך לשני. המבנה הוא:

[שם A] の [שם B] = "B של A" / "B מסוג A"

שם A בא לפני の ומגדיר את B. בעברית לרוב נתרגם "של" — אבל לא תמיד. לפעמים A מציין סוג או נושא של B:

私の本 （わたしのほん）
הספר שלי ← שייכות
田中さんの猫 （たなかさんのねこ）
החתול של טנקה ← שייכות
日本語の本 （にほんごのほん）
ספר ביפנית ← סוג / נושא
学校の先生 （がっこうのせんせい）
מורה בבית-הספר ← שייכות למוסד

אפשר גם לשרשר: A の B の C — C של B של A:

私の学校の先生 （わたしのがっこうのせんせい）
המורה של בית-הספר שלי

שימוש עם משפט שלם: מחברים の ל-は כמקודם ומוסיפים את הנשוא:

これは私の本です。 （これはわたしのほんです）
זה הספר שלי.
あれは田中さんの車ですか。 （あれはたなかさんのくるまですか）
האם זה (שם) המכונית של טנקה?
טיפ חשוב: שם A לפני の הוא תמיד שם עצם — לא שם תואר. שמות תואר מחוברים אחרת (נלמד בשבוע הבא).

（これはにほんごのほんですか）

איך בונים מספרים מורכבים?

פשוט מכפילים: ספרה × יחידה, ומחברים. אין מילת "אחד" לפני 百 ו-千 בדיבור רגיל — אומרים ひゃく (לא いちひゃく) ו-せん (לא いちせん). אבל לפני まん כן: いちまん.

十一←11 = じゅう (10) + いち (1)
二十←20 = に (2) + じゅう (10)
三十五←35 = さん + じゅう + ご
二百←200 = に (2) + ひゃく (100)
三千←3,000 = さん + せん
一万五千←15,000 = いちまん + ごせん
שים לב — שינויי הגייה! כמה מספרים משנים הגייה בחיבור:
300 = さんびゃく (לא さんひゃく)
600 = ろっぴゃく · 800 = はっぴゃく
3,000 = さんぜん · 8,000 = はっせん
כל שאר המקרים רגילים.

נושא 3

מחיר ורכישה: いくら ו-円（えん）

כדי לשאול "כמה עולה?", משתמשים ב-いくら ("כמה / מה המחיר"). המטבע היפני הוא 円（えん） — ין. המבנה הוא:

[חפץ] は いくらですか。 = "כמה עולה [חפץ]?"
[מחיר] 円です。 = "עולה [מחיר] ין."
この本はいくらですか。 （このほんはいくらですか）
כמה עולה הספר הזה?
五百円です。 （ごひゃくえんです）
עולה 500 ין.
あのカメラはいくらですか。 （あのかめらはいくらですか）
כמה עולה המצלמה ההיא?
三万二千円です。 （さんまんにせんえんです）
עולה 32,000 ין.
